Public Instance methods

cn()

Calculates de checksum to use as ‘ch’ parameter on request_uri, for example the checksum of www.rubyonrails.com is 63157409018

page_rank(uri = @uri)

Return a number between 0 to 10, that represents the Google PageRank

request_uri()

URI that gets the pagerank, for example to get the Google‘s pagerank of www.rubyonrails.com the request_uri is toolbarqueries.google.com/search?client=navclient-auto&hl=en&ch=63157409018&ie=UTF-8&oe=UTF-8&features=Rank&q=info:http%3A%2F%2Fwww.rubyonrails.com
